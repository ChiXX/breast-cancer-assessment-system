# 化疗副作用分诊技能手册

本技能用于对化疗后患者报告的副作用进行分级评估，提供精准的处置建议。

## 症状-风险映射索引

| 症状类别 | 风险等级 | 资源文件 |
|---------|---------|---------|
| 神经系统急症（剧烈头痛、单侧肢体无力、视力模糊） | HIGH | [neurological-emergency.md](./neurological-emergency.md) |
| 心脏毒性/肺栓塞（胸闷、气促、胸痛） | HIGH | [cardiotoxicity.md](./cardiotoxicity.md) |
| 出血性并发症（鼻出血、皮肤瘀青/瘀点） | HIGH | [bleeding.md](./bleeding.md) |
| 口腔黏膜炎（口腔溃疡、疼痛） | MEDIUM | [oral-mucositis.md](./oral-mucositis.md) |
| 癌因性疲劳（严重乏力、影响自理） | MEDIUM | [fatigue.md](./fatigue.md) |
| 指甲改变（变色、横纹、变脆） | LOW | [nail-changes.md](./nail-changes.md) |

## 使用说明

1. 根据患者主诉的症状类别，查阅对应的子资源文件
2. 子资源文件中包含详细的评估逻辑 JSON 和处置建议
3. 高风险（HIGH）症状需立即线下就医；中风险（MEDIUM）需联系团队或密切观察；低风险（LOW）可居家护理
4. 所有评估基于 CTCAE v5.0 标准
