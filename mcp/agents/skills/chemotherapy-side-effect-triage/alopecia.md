# 化疗相关脱发评估与护理

## 概述

脱发是许多乳腺癌化疗方案（尤其含紫杉醇、蒽环类药物）的常见副作用，通常在化疗开始后2-3周出现，在整个化疗周期内持续，但化疗结束后3-6个月头发通常会重新生长。这属于低风险情况，无需紧急处理。

## 护理要点

- 使用温和无硫酸盐洗发水，温水轻柔清洗
- 使用软齿宽梳，减少梳理频率
- 选择丝质或棉质枕套减少摩擦
- 外出时可使用太阳帽或头巾保护头皮（化疗期间头皮对紫外线较敏感）
- 如有意愿，可提前选购假发
- 继续记录掉发情况并在常规复诊时告知医生
