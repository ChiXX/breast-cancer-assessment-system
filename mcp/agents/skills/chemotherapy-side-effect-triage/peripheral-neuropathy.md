# 化疗相关周围神经病变（手脚麻木）评估与处置

## 概述

化疗引起的周围神经病变（手脚麻木）可能是严重的副作用，需要立即评估。突发严重的周围神经症状需按高风险处理并立即就医。

## 处置要点

- **立即就医**：前往医院神经内科或肿瘤内科进行详细检查
- **暂停化疗**：暂时停止当前的化疗计划，直到医生评估完毕
- **紧急联系**：立即联系医疗团队（contact_team: true）
- 排除周围神经病变的可能性
