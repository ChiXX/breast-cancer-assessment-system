# 化疗相关皮肤干燥/瘙痒/色素改变评估与护理

## 概述

皮肤干燥、瘙痒和色素改变是化疗常见的轻度副作用，通常无需特殊医疗干预。属于低风险情况，继续观察并记录变化即可。

## 护理要点

- 每日洗澡后立即（3分钟内）涂抹无香精、无酒精的温和保湿乳液（如维生素E乳、凡士林基保湿霜）
- 洗澡水温不宜过高，时间控制在15-20分钟内
- 外出时使用SPF30以上防晒霜（化疗期间皮肤对光线更敏感）
- 选择棉质宽松衣物减少摩擦
- 皮肤色素改变通常在化疗结束后数月内逐渐恢复
- 如出现皮肤破损、感染或严重皮疹，请联系医疗团队
