# 口腔黏膜炎（化疗后）

## 概述

化疗后口腔黏膜炎（Oral Mucositis）是常见的化疗副作用，通常在化疗后5-14天出现。表现为口腔黏膜红肿、溃疡、疼痛，严重时可影响进食和说话。

## 关键处置要点

- **联系团队**：中风险情况，建议尽快联系肿瘤科护理团队获取处方级漱口液或止痛方案
