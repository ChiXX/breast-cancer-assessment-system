# 化疗后指甲改变管理

## 风险等级：低风险

### 评估标准
- 症状：指甲变色（发黑）、变脆、出现横纹
- CTCAE Grade 1
- 常见于紫杉类药物治疗后
- 通常在化疗结束后6-12个月随新指甲生长逐渐恢复

### 处置建议

| 项目 | 内容 |
|------|------|
| **风险等级** | 低风险 |
| **下一步建议** | 继续观察与记录，常规复诊时告知医生。护理建议：修剪指甲时保持清洁，用圆头剪刀修剪成圆弧形；可使用硬化型护甲油（避免含甲醛成分）保护指甲；避免长时间浸水或接触化学清洁剂，做家务时戴橡胶手套；避免咬指甲或用力剥除受损甲片 |
| **是否建议联系团队** | 否（但若指甲周围出现红、肿、热、痛或化脓等甲沟炎迹象，则需及时联系） |
| **参考依据** | QA-L-005 |

### 注意事项
- 若指甲周围出现红、肿、热、痛或化脓等甲沟炎迹象，需及时联系医疗团队
- 需询问指甲变色从第几次化疗后开始出现
