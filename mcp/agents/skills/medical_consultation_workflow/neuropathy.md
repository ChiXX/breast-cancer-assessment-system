# 化疗后周围神经病变(CIPN)管理

## 风险等级：中风险

### 评估标准
- 症状：手脚麻木、刺痛、灼热感或感觉减退
- 常见于紫杉类、长春碱类或铂类药物治疗后
- 若继续进展可能造成不可逆的神经损伤

### 处置建议

| 项目 | 内容 |
|------|------|
| **风险等级** | 中风险 |
| **下一步建议** | 尽快联系医疗团队报告此症状；注意安全防护（洗澡时用肘部试水温防烫伤，穿防滑平底鞋，保持地面整洁防跌倒）；医生可能会评估是否需要调整化疗剂量 |
| **是否建议联系团队** | 是 |
| **参考依据** | QA-M-003 |

### 注意事项
- 需确认麻木感的起始时间及进展方式以完善评估
- 若出现运动功能障碍（如握力下降、行走不稳），需升级为高风险处置
