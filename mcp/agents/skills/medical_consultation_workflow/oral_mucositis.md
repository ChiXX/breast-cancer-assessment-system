# 化疗后口腔黏膜炎管理

## 风险等级：中风险

### 评估标准
- 症状：口腔内出现多个溃疡
- 常见于化疗后5-14天
- 需专业护理指导以防加重

### 处置建议

| 项目 | 内容 |
|------|------|
| **风险等级** | 中风险 |
| **下一步建议** | 尽快联系肿瘤科护理团队。日常护理：每日用无酒精漱口水（如生理盐水或碳酸氢钠溶液）漱口4-6次；避免过热、过酸、辛辣和硬质食物；选择软烂、温凉的流食（如粥、豆腐、冷藏果汁） |
| **是否建议联系团队** | 是 |
| **参考依据** | QA-M-002 |

### 注意事项
- 若疼痛加重到无法喝水，需立即联系团队
- 需确认患者是否正在接受化疗治疗
